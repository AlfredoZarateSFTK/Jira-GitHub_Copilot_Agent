Como equipo DevOps / Engineering
quiero explorar la nueva funcionalidad de GitHub Copilot Coding Agent integrada con Jira
para entender sus capacidades, costos y beneficios en nuestro flujo de desarrollo.

Descripción
La nueva funcionalidad en preview permite asignar issues de Jira directamente a GitHub Copilot, el cual puede analizar el ticket, generar código y crear automáticamente un draft Pull Request en el repositorio de GitHub.

El agente utiliza el contexto del ticket (descripción, comentarios y etiquetas) para generar una propuesta de solución y ejecutar tareas de desarrollo de forma autónoma dentro del pipeline de desarrollo.

Esta exploración busca evaluar cómo esta integración puede acelerar la resolución de tickets, mejorar la productividad del equipo y fortalecer los flujos DevOps impulsados por AI.

Objetivos de la exploración
Entender cómo funciona la asignación de issues de Jira al Copilot Agent.

Validar cómo se genera automáticamente código y Pull Requests.

Identificar beneficios en productividad del flujo Jira → GitHub → CI/CD.

Analizar costos y licenciamiento necesario.

Identificar posibles casos de uso dentro de nuestros proyectos.

Alcance de la exploración
Evaluar:

Flujo Jira → Copilot → Pull Request

Automatización de tareas de desarrollo

Uso de Agent Mode para resolver issues

Integración con repositorios existentes

Impacto en el ciclo de desarrollo

Criterios de aceptación
La exploración se considera completada cuando:

Se entiende cómo asignar un issue de Jira al Copilot Coding Agent

Se valida si Copilot puede:

generar código automáticamente

crear Pull Requests

proponer soluciones técnicas

Se documentan beneficios potenciales para DevOps

Se documentan limitaciones o prerequisitos

Se identifican costos aproximados de licenciamiento

Beneficios esperados
Reducción del tiempo para resolver tickets

Automatización de tareas repetitivas de desarrollo

Mayor velocidad en generación de código y PRs

Mejora del flujo DevOps + AI

Estudios y análisis muestran que herramientas como Copilot pueden reducir significativamente tiempos de desarrollo y revisión de código, permitiendo a los ingenieros enfocarse en tareas de mayor valor.

Costos aproximados (referencia)
Planes comunes de GitHub Copilot:

Copilot Business: ~ $19 USD / usuario / mes

Copilot Enterprise: ~ $39 USD / usuario / mes

Las funcionalidades de Agent Mode y Coding Agent consumen “premium requests” dependiendo del modelo utilizado.
